# vishalbagla.com

Personal landing page for **Vishal Bagla** — VP Product Management, AI PM Leader, Creator & Educator.

## About

A single-file static landing page built with pure HTML and CSS (no frameworks). Designed to project expertise at the intersection of AI and Product Management.

## Sections

- **Hero** — Positioning statement, key stats, animated role cycler
- **About** — Career narrative and credentials
- **AmbitionBox** — Dedicated section on 4+ years of product leadership at Info Edge
- **Career Timeline** — Vertical journey from 2008 to present
- **Thought Leadership** — LinkedIn reach, creator rankings, brand collaboration case studies
- **Newsletter** — Substack CTA (vishalbagla.substack.com)
- **Work With Me** — Speaking, advisory, content partnerships, mentoring
- **Connect** — LinkedIn, Substack, Topmate

## Tech

- Pure HTML + CSS + vanilla JS
- No build step, no dependencies
- Scroll-reveal animations via Intersection Observer
- Fully responsive

## Deployment

Hosted on [vishalbagla.com](https://vishalbagla.com). To run locally:

```bash
python3 -m http.server 8080
# then open http://localhost:8080
```
